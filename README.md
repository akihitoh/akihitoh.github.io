## 研究テーマ
- **ニューラルネットワークによるシミュレーション代理（サロゲート）モデル**: 歩行者流制御施策の最適化、自動運転通信の性能評価、電波伝搬マップ、衛星フェーズドアレイのカバレッジ推定
- **スマートシティ向け最適化とサービス実装**: 避難経路計画（線形計画・Transformer誘導）、介護タクシー配車、Pub/SubによるIoTメッセージ配信
- **3次元点群・UAVを用いた空間/人物センシング**: 高密度群衆のトラッキング・人数推定、UAV点群からの建物認識と異常検知、ミリ波レーダーによるオクルージョン下の人物検出
- **車両・交通センシングと安全支援**: スマートフォンセンシングによる危険交通状況検出、高速道路合流点予測、交通映像からの車線交通量抽出
- **緊急・耐災害ネットワーク**: LoRaを用いた緊急ネットワークのスケジューリング、災害時の通信・情報集約基盤、電子トリアージシステム
- **大規模ネットワーク解析・シミュレーション**: Network Calculus に基づく車載ネットワークの遅延評価、無線相互干渉調停、密に設置される路側機/基地局群のスケジューリング
- **スマート建物・人の行動に基づくエネルギー制御**: 生活ノウハウのモデル化、温熱快適性を考慮した空調制御、タスク/アンビエント照明
- **人の動きを考慮したシステム設計**: 人流・歩行者トラッキング、モビリティモデル、群衆センシング
