# ソフトウェア・データセット

このページでは、研究で開発したソフトウェアツールやデータセットを公開しています。研究利用に際しては、引用元論文の参照をお願いします。

## ソフトウェアツール

### MobiReal
- **概要**: モバイルアドホックネットワーク(MANET)のシミュレータで、現実的な人の移動パターンと地理的制約を考慮したネットワーク性能を評価できます。
- **主な機能**:
  - 地理情報と人の行動パターンを統合した現実的なモビリティモデル
  - ノード密度分布の制御と自然な移動パターン生成
  - ネットワークシミュレーションとの統合
- **開発言語**: C++, Java
- **関連論文**: 「A Simple Mobility Model Realizing Designated Node Distributions and Natural Node Movement」（IEEE MASS 2011）
- **[ダウンロードページ](https://github.com/mobireal)**

### HumanS
- **概要**: 都市環境における人の移動と行動を詳細にモデル化・シミュレーションするツールです。特にセンサ配置最適化や人流推定に活用できます。
- **主な機能**:
  - 多様な都市環境のモデリング
  - 人の行動パターンの確率的生成
  - センサによる検知シミュレーション
  - 可視化インターフェース
- **開発言語**: Java, Python
- **関連論文**: 「HumanS: A Human Mobility Sensing Simulator」（NTMS 2012）
- **[ダウンロードページ](https://github.com/humans-sim)**

### D-Sense
- **概要**: センサネットワーク開発のためのデータ中心型プログラミング環境です。センサデータの収集・処理・分析を統合的に扱えます。
- **主な機能**:
  - データフローベースのプログラミングモデル
  - センサーノード間の協調処理
  - リアルタイムデータ処理・可視化
- **開発言語**: C, Python
- **関連論文**: 「D-Sense: An Integrated Environment for Algorithm Design and Protocol Implementation in Wireless Sensor Networks」（IFIP/IEEE MMNS 2008）
- **[ダウンロードページ](https://github.com/dsense)**

## データセット

### 大阪大学キャンパス歩行者トラジェクトリデータ
- **概要**: 大阪大学キャンパス内の主要な通路における歩行者の移動軌跡データです。1日を通して測域センサで計測した点群データから抽出された人の移動軌跡を含みます。
- **特徴**:
  - 測域センサによる高精度三次元点群データ
  - 混雑度の異なる時間帯のサンプル
  - 30日間の長期データ収集
  - 位置情報と時刻のタイムスタンプ
- **形式**: CSV, JSON
- **サイズ**: 約1.2GB（圧縮時）
- **関連論文**: 「公共空間における三次元点群の不完全性に対して堅牢な歩行者トラッキング手法」（情報処理学会論文誌 2022）
- **[ダウンロードページ](https://github.com/pedestrian-dataset)**

### 路線バス乗降データ
- **概要**: 実際の路線バスにおける乗降客数と時刻を測域センサーを用いて計測したデータセットです。公共交通の利用状況分析や予測モデル構築に活用できます。
- **特徴**:
  - 乗降客数の時系列データ
  - バス停ごとの乗降パターン
  - 平日・休日の差異データ
  - 気象情報との紐づけ
- **形式**: CSV
- **サイズ**: 約200MB
- **関連論文**: 「測域センサを利用した高精度な路線バス乗降計測システム」（情報処理学会論文誌 2019）
- **[ダウンロードページ](https://github.com/bus-dataset)**

### 都市道路交通流データ
- **概要**: 交通カメラから抽出した車両の流れを含む都市交通データセットです。交差点や主要道路での車両の速度、密度、流量などの情報を含みます。
- **特徴**:
  - 複数地点の同期計測データ
  - 車種別の集計
  - 長期間（3ヶ月）のデータ
  - 5分間隔のサンプリング
- **形式**: CSV, HDF5
- **サイズ**: 約3GB
- **関連論文**: 「数理的な移動軌跡モデルに基づく広域交通流の推定手法」（情報処理学会論文誌）
- **[ダウンロードページ](https://github.com/traffic-dataset)**
